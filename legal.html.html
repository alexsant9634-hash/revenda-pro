O Revenda Pro leva sua privacidade a sério. Este documento explica de forma clara quais dados coletamos, como usamos e como protegemos as informações suas e das suas clientes.

1. Quem somos

O Revenda Pro é um aplicativo de gestão para revendedoras de cosméticos, desenvolvido e operado por Alex dos Santos Barbosa, pessoa física, com base em Maracanaú, Ceará, Brasil.

Contato: alexsant9634@gmail.com

2. Dados que coletamos

2.1 Dados da revendedora (você)

  • Nome completo — fornecido no cadastro
  • Endereço de e-mail — usado para login e comunicações
  • Senha — armazenada de forma criptografada, nunca em texto puro
  • Telefone e chave Pix — fornecidos voluntariamente nas configurações
  • Dados de pagamento — processados pelo Stripe, nunca armazenados diretamente por nós

2.2 Dados que você cadastra no app

  • Nome e WhatsApp das suas clientes
  • Histórico de pedidos e valores
  • Produtos, preços e estoque
  • Fornecedores e despesas

Esses dados pertencem a você. Nós os armazenamos para que o app funcione e são acessíveis apenas pela sua conta.

2.3 Dados técnicos coletados automaticamente

  • Tipo de dispositivo e navegador
  • Data e hora de acesso
  • Endereço IP

3. Como usamos os dados

  • Fornecer e melhorar as funcionalidades do app
  • Processar pagamentos e gerenciar assinaturas
  • Enviar comunicações sobre o serviço (novidades, alertas)
  • Cumprir obrigações legais

Não vendemos, alugamos ou compartilhamos seus dados com terceiros para fins comerciais.

4. Com quem compartilhamos dados

Supabase

Nosso banco de dados e sistema de autenticação. Os dados são armazenados em servidores na região de São Paulo (Brasil). Política de Privacidade do Supabase.

Stripe

Processador de pagamentos. Recebe apenas os dados necessários para processar sua assinatura (e-mail e dados do cartão). Política de Privacidade do Stripe.

Vercel

Hospedagem da nossa API. Pode processar dados técnicos das requisições. Política de Privacidade do Vercel.

5. Seus direitos (LGPD)

De acordo com a Lei Geral de Proteção de Dados (Lei nº 13.709/2018), você tem direito a:

  • Acesso — saber quais dados temos sobre você
  • Correção — corrigir dados incompletos ou incorretos
  • Exclusão — solicitar a exclusão dos seus dados
  • Portabilidade — receber seus dados em formato legível
  • Revogação do consentimento — cancelar o uso dos seus dados
  • Oposição — se opor ao tratamento dos seus dados

Para exercer qualquer direito, entre em contato: alexsant9634@gmail.com

6. Dados das suas clientes

Ao cadastrar nome e WhatsApp das suas clientes no app, você assume a responsabilidade de:

  • Ter o consentimento delas para armazenar esses dados
  • Usar as informações apenas para fins relacionados à sua atividade de revenda
  • Não compartilhar esses dados com terceiros

O Revenda Pro age como operador de dados nesse caso — armazenamos e processamos conforme suas instruções, mas a responsabilidade legal perante as clientes é sua.

7. Segurança dos dados

  • Senhas criptografadas com bcrypt via Supabase Auth
  • Comunicação criptografada via HTTPS/TLS
  • Row Level Security (RLS) — cada conta só acessa seus próprios dados
  • Chaves de API protegidas no servidor, nunca expostas ao cliente

8. Retenção de dados

Seus dados são mantidos enquanto sua conta estiver ativa. Ao cancelar e solicitar exclusão, excluímos seus dados em até 30 dias, exceto quando obrigados por lei a manter.

9. Cookies e armazenamento local

O app usa localStorage do navegador para armazenar dados temporários e melhorar a performance offline. Não usamos cookies de rastreamento ou publicidade.

10. Alterações nesta política

Podemos atualizar esta política periodicamente. Notificaremos por e-mail em caso de mudanças significativas. O uso continuado do app após a notificação implica aceitação das mudanças.
